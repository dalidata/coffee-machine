🌟 Projet Python : Gestion Automatisée d'une Machine à Café ☕

Fonctionnalités clés :

Gestion des pièces insérées : L'application suit et valide les montants insérés par l'utilisateur.

Menu interactif : Les utilisateurs peuvent choisir leur café préférée parmi une sélection de options.

Rapports en temps réel :

📊 Ressources disponibles : Suivi des ingrédients (café, lait, eau, etc.) et alertes en cas de pénurie.

💰 Total des ventes : Calcul automatique des revenus générés par la machine.

Interface utilisateur conviviale : Développée avec Streamlit, l'application offre une expérience fluide et moderne.

Technologies utilisées :

🐍 Python pour la logique métier et les calculs.

🎨 Streamlit pour créer une interface web interactive et responsive.

📦 Gestion des données avec des structures Python (listes, dictionnaires) pour suivre les ressources et les transactions.
